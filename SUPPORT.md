# Support

Use the following paths:

- bug report: reproducible defects in current behavior
- feature request: product or workflow improvements
- security report: follow `SECURITY.md`, do not post publicly first
- contribution question: open a GitHub discussion or issue with exact context and command

Please include:

- exact command
- profile name/path
- output mode
- expected vs actual behavior
- whether the issue appears in `--json` diagnostics
